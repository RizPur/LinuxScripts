# Productivity Scripts

This directory contains a collection of personal scripts for managing tasks and time.

## Scripts

- **`pomodoro.py`**
  A simple command-line Pomodoro timer. Run it to start a focus session.

- **`tasks.py`**
  A command-line task manager. Use it to add, view, and manage your to-do list.

- **`agenda.py`**
  A tool for managing your daily agenda and schedule.

- **`today.sh` & `workflows.sh`**
  Shell scripts for custom, personal workflows.
